## Introduction
The LinkedIn Sales Nav Profile Converter Scraper allows you to take LinkedIn Sales Navigator profiles and convert them into regular LinkedIn profiles with 95% accuracy. It works by processing a CSV file containing essential data (first name, last name, title, company, location) and automating the conversion process for each entry.

### What It Helps You Do
- Convert bulk LinkedIn Sales Navigator profiles into standard profiles.
- Handle CSV uploads with data points like names, company info, and location.
- Automate the conversion process to save time on manual data entry.
- Integrate this tool into lead generation or CRM workflows.

## What Data This Scraper Extracts
| Field Name | Field Description |
|------------|-------------------|
| firstName | First name of the LinkedIn contact. |
| lastName | Last name of the LinkedIn contact. |
| title | Job title of the contact. |
| company | Name of the company the contact works for. |
| location | Geographical location of the contact. |
| linkedinProfileUrl | URL of the converted regular LinkedIn profile. |
